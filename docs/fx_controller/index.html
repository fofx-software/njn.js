fofx.Controller

A module for building interactive web apps and DRYing your HTML with powerful templating.

Quick Start

Download the code:

fofx.js

fx_controller.js

Put it in your HTML:


<head>
  <script src="fx.js"></script> <!--run fx.js first-->
  <script src="fx_controller.js></script>
</head>
syntax highlighting courtesy of Prism

Put it to work:


<body>
  <div fx-controller="example-controller">
    {{exampleText}} <!-- will be processed as 'hello world' -->
  </div>
  <script>
    fofx.controller('example-controller', {
      exampleText: 'hello world'
    });
  </script>
</body>

Usage

Initialize the fofx.Controller:

Initialize an instance of fofx.Controller: 
fofx.controller();
The first argument gives the new fofx.Controller a name so you can reference it in your HTML: 
fofx.controller('example-controller');
The second argument provides the fofx.Controller's viewInterface: 

fofx.controller('example-controller', {
  exampleText: 'hello world'
});

Reference the fofx.Controller:

Reference the fofx.Controller by its name in an fx-controller attribute: 

<div fx-controller="example-controller"></div>



fofx.controller('example-controller');

Access the viewInterface:

The properties of the viewInterface can be accessed from the HTML: 

<div fx-controller="example-controller">{{exampleText}}</div>
<script>fofx.controller('example-controller', { exampleText: 'hello world!' });</script>

Referencing a method in the viewInterface

When the property is a function, in normal cases it is called without any arguments: 

<div fx-controller="example-controller">
  {{getText}} <!--processes to 'my text'-->
</div>
<script>
  fofx.controller('example-controller', {
    getText: function(arg1) { if(typeof arg1 === 'undefined') return 'my text'; }
  });
</script>

Referencing an object in the viewInterface


<div fx-controller="example-controller">
  <span fx-context="subObj">
    {{myText}} <!--processes to "it's me!"-->
  </span>
</div>
<script>
  fofx.controller('example-controller', {
    subObj: { myText: 'it's me!' }
  });
</script>
